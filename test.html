Staff Goals & Expectations

This is an in-depth documentation of goals and expectations that come with having a certain staff role in our community. While the term "expectations" may sound a bit formal, it's important to keep in mind that everyone with a staff role is just a volunteer and that this list is merely a clear overview of what each role entails to encourage staff and server activity!

In-game & Discord Commands

Dyno Command Syntax and Usage

Dyno is a bot we use to help make Discord server management and moderation easier. Below is a comprehensive list of Dyno commands that are available for our staff to use. Command permissions are inclusive of preceding commands as the list progresses (ex. moderators have both moderator permissions and trainee permissions).

  • Trainee Moderator Commands:
    • ?modlogs (user id) (check logs for user moderation history)
    • ?note (user id) (note) (leave notes on a user for future mods to reference)
    • ?notes (user id) (view all previous notes on a user)
    • ?warn (user id) (reason) (warn the player of a rule violation)
    • ?mute (user id) (time) (reason) (start muting after 3 warnings)
  • Moderator Commands:
    • ?delnote (user id) (note id) (leave notes on a user for future mods to reference)
    • ?kick (user id) (reason) (removes the user from the Discord server)
  • Senior Moderator Commands:
    • ?ban save (user id) (time) (reason) (permanently removes the user from the Discord server and saves all their messages)
    • ?ban (user id) (time) (reason) (permanently removes the user from the Discord server and deletes all their messages)
      • ‚ö†NOTICE: DO NOT use this command unless the user is spamming NSFW and all messages ever posted by the user are best removed.
    • ?unban (user id) (optional reason) (unban a member from the Discord server)
